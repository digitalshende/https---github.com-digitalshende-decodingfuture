Data Science FAQs

Data Science is an interdisciplinary field that uses statistics, algorithms, machine learning, and data analysis techniques to extract meaningful insights from structured and unstructured data.

The key components of Data Science include:
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Machine Learning & AI

The most commonly used languages are:
1. Python (most popular due to its vast libraries)
2. R (commonly used in statistical analysis)
3. SQL (for database management)

1. Data Scientist (Builds ML models & analyzes data)
2. Machine Learning Engineer (Deploys ML models in production)
